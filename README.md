# Paisabazaar-Banking-Fraud-Analysis-Dashboard-Excel-
Excel-based Banking Fraud Analytics Dashboard analysing customer credit behaviour, fraud indicators, credit utilization patterns, and financial risk profiles to support fraud detection and risk management.

📌 Project Overview

This project presents a Banking Fraud Detection Analytics Dashboard built using Microsoft Excel.

The dashboard analyses customer financial data to identify high-risk customers, fraud indicators, and credit behaviour patterns.

The objective of this project is to demonstrate financial fraud analytics, customer risk analysis, and data-driven fraud detection using Excel dashboards.

📊 Key Business Metrics

Total Customers: 100,000

High-Risk Customers: 54,058

Fraud Rate: 54.06%

Average Annual Income: ₹50,505

Average Outstanding Debt: ₹1,426

Average Credit Utilization: 32.28%

These metrics provide an overview of customer financial behaviour and fraud exposure.
